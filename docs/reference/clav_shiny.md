# Run the Shiny application.

Run the Shiny application.

UI for cluster analysis application

Server function for cluster analysis application

## Usage

``` r
cluster_shiny(...)

clav_shiny_ui()

clav_shiny_server(input, output, session)
```

## Arguments

- ...:

  named list of data frames to make available in the Shiny application.

- input:

  Shiny input object.

- output:

  Shiny output object.

- session:

  Shiny session object.
